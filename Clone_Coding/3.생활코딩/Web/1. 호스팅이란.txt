#호스팅이란?
대형 서버의 기능을 빌려쓰는것을 의미

#호스팅 종류
1) 웹호스팅(서버를 운영하기 위한 모든것을 제공) : 웹서버, 미들웨어, 데이터 베이스
-> 하나의 서버를 여러명이 공유하여 사용하는 것
(공유자 중 한명이 많은 트래픽을 잡아먹는다면 서버가 다운되는 제약이 있을 수 있다.)

2) 서버호스팅
-> 한명의 고객이 하나의 서버장비를 임대하는 호스팅
(주로 대형 사이트를 구축할때 사용되며 서버를 단독으로 사용하기 때문에 설치, 삭제등의 개발이 자유롭다.)

3) 클라우드호스팅
물리적 서버장비가 아닌 가상 서버를 임대하는 방식. 
(이용한 만큼 금액을 지불하는 방식이고 일시적인 트래픽이 많은 사이트에 적합한 호스팅이다.)
