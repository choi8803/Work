# github - 가입 및 파일 업로드

1) Create an account
2) Sign in
3) New Repository
4) Add file 
-> 위 과정을 통해 버전별로 소스 관리가 가능하다.


# github - Server 사용방법(Web hosting)

1) Settings
2) > GitHub Pages
3) > Source : "None" -> main" 변경 
4) > Source > Save
-> GitHub Pages에 "Your site is published at https://choi8803.github.io/Web1/"으로 표시된다.
(위 주소로 사이트가 생성된다.)

- My (create -> index.html) : Web Browser를 통해 github에 파일을 업로드
                                              
- Web Browser (요청 : index.html) <--> Web Server (응답 : http://choi8803.github.io/Web1)
