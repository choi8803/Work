# Lambda(람다식)

목적
1. 간단하고 재활용이 필요 없는 함수를 만들 때 사용
   - 함수 및 변수의 이름을 고민 할 시간을 줄임.
2. 별도로 함수를 추가 하지 않고 함수 안에서 호출해서 사용
3. 무명 메서드에서 발전 된 형태
4. 함수의 인자 값이 메서드 형태일 경우 주로 사용 됨.

+ C#에서는 delegate 무명 메서드를 Generic 형태로 정의해 놓음
 - Func<T, T Result> : 반환 형이 있는 delegate Type
 - Action<T> : 반환 형이 없는 delegate Type(Void)

(입력 파라미터) => { 문장 블록 };
